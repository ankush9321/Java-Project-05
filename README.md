# Bank Management System (Java, JPA, Hibernate, PostgreSQL)
<h6>Overview</h6>
<p>This is a simple console-based Bank Management System implemented in Java using JPA (javax.persistence) and Hibernate with PostgreSQL as the database. 
It supports account creation, deposits, withdrawals, and balance checks, with PIN-based authentication for security. 
The project also uses Hibernate second-level caching to improve performance for frequently accessed accounts.</p>
